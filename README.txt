# MtGLoL

Custom Magic the Gathering set based on League of Legends.

Process of merging and exporting all sets to Cockatrice:
1) Make edits to any .mse-set files under 'Cards\Card Sets' and save.
2) Run merge.jar
3) Open output.mse-set
  a) export HTML into 'Cockatrice\data\customsets\lol.xml'
  b) export all images into 'Cockatrice\data\pics\CUSTOM'
4) Restart Cockatrice.
